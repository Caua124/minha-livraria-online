# 📚 Livraria Online

Projeto de uma livraria online desenvolvido para permitir que usuários naveguem, pesquisem e comprem livros de forma simples e intuitiva.

---

## 🚀 Funcionalidades

* 🔍 Busca de livros por título, autor ou categoria
* 📖 Visualização de detalhes do livro
* 🛒 Carrinho de compras
* 💳 Finalização de pedidos (checkout)
* 👤 Cadastro e autenticação de usuários
* ⭐ Avaliações e comentários de livros

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML, CSS, JavaScript (ou React, Vue, etc.)
* **Backend:** Node.js / Python / Java
* **Banco de Dados:** MySQL / PostgreSQL / MongoDB
* **Outros:** API REST, JWT para autenticação

---

## 📂 Estrutura do Projeto

```
livraria-online/
```

## ⚙️ Instalação e Execução

### Pré-requisitos

* Node.js (ou outra stack utilizada)
* Gerenciador de pacotes (npm, yarn, pip, etc.)
* Banco de dados configurado

### Passos

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/livraria-online.git

# Acesse a pasta
cd livraria-online

# Instale as dependências
npm install

# Execute o projeto
npm start
```

---

## 🔑 Variáveis de Ambiente

Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:

```
DB_HOST=localhost
DB_USER=seu_usuario
DB_PASS=sua_senha
JWT_SECRET=sua_chave_secreta
```
